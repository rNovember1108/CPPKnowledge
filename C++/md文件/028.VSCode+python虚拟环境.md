# VS Code python虚拟环境

## 安装python

略

## 安装虚拟环境

    pip install virtualenv

## 创建虚拟环境

    python -m venv env

## 激活虚拟环境

    .\env\Scripts\activate

## 关闭虚拟环境

    deactivate

## 安装包

    pip install xxxxx

## 卸载包

    pip uninstall xxxxx

## 展示已安装包

    pip list

## 问题

在终端出现报错：无法加载文件\venv\Scripts\activate.ps1，因为在此系统上禁止运行脚本。

1. 终端输入**get-executionpolicy**，回车返回Restricted

2. 复制Windows Powershall在windows以管理员的身份运行，命令窗口输入**set-executionpolicy** **remotesigned**，回车，输入Y

3. 终端写入**get-executionpolicy**命令回车，返回remotesigned

4. 输入 .\env\Scripts\activate开启虚拟环境
